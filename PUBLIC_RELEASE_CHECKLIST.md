# Public release checklist

This package is prepared for a public GitHub repository.

- [x] Source code and documentation are in English.
- [x] Full research workbooks are excluded.
- [x] Synthetic demonstration workbooks are included.
- [x] API keys and `.env` files are excluded.
- [x] Generated outputs, logs, caches, and local paths are excluded.
- [x] Personal names, email addresses, and institutional identifiers are excluded.
- [x] A license, citation file, dependency files, tests, and reproduction script are included.

Before publishing, replace the generic repository URL and citation metadata only when the final public repository and associated article are available.
